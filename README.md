
Aide: Giving a helping hand
===========================

<!-- README.md is generated from README.Rmd. Please edit that file -->
Contains simple functions commonly used in data analysis. For instance, for average and median

``` r
library(aide)
average(1:10)
#> [1] "5.5"
typeof(average(1:10))
#> [1] "character"
med(1:10)
#> [1] "5.5"
typeof(med(1:10))
#> [1] "character"
```
